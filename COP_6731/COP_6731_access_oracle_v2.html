Log in to the Engineering Desktops

 

 

 

 

 

 

Access the terminal from the windows desktop (in VMWare)

SSH into oraclelinux.eng.fau.edu

At the prompt type ssh your_user_name@oraclelinux.eng.fau.edu

your_user_name is the name for your FAU email address, i.e. rraven2021@fau.edu

 

 

 

Default password is oracle

Enter Oracle database server with SQL*Plus command

At prompt type sqlplus your_user_name

Default password is oracle

Create row

Exit to VMware desktop

Find spool file in VMware desktop

This PC -> Local Disk (C) -> Users -> your_user_name

Right click and select Notepad ++ to view file

Or double click to view in Notepad
